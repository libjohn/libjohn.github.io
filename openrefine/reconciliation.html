5.1 Hand’s On

  1. Open https://raw.githubusercontent.com/libjohn/openrefine/master/data/AA-authors-you-should-read.csv

  2. authors > Edit column > Add column based on this column… ; Name = “Reconciled Authors”

  3. authors > Reconcile > Start reconcilling…

    1. One Time…
    2. documentation: http://refine.codefork.com/
    3. Click: Add Standard Service ...
    4. From the refine.codefork.com, use VIAF http://refine.codefork.com/reconcile/viaf And if you use it a lot, read the documentation follow the install instructions: https://github.com/codeforkjeff/refine_viaf
    5. Rest of Time…
    6. VIAF > Person > Start Reconciling
    7. reconciled authors > Reconcile > Actions > Match each cell to it’s best candidate
    8. Choose New Match for James Baldwin
    9. Choose new match for Richard Wright > Create new item
  4. Can Export as CSV (Excel, etc.) with Reconciled info.

  5. Remove all facets

  6. Convert 12PM to 00PM
    • date2 Edit cells > transform
      • Expression = value.replace(" 12:"," 00:")
  7. Now make a date facet
    • date2 > Facet > Timeline facet

5.2 Lipsum More Foo Text

Goal: Make the same data transformations as in the previous slide’s steps 2-7, but this time you’ll do it in one step. Think about how GREL allows you to take shortcuts.

  1. issue_date > Edit Column > Add column based on this column…
    • New Column Name = date3
    • Expression = if(value.slice(20,22) == "PM", value.replace(/(.*\s)12(:\d\d:.*)/,"$100$2"), value).toDate()
  2. Now make a date facet
    • date3 > Facet > Timeline facet
